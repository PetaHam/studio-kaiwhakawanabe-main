# **App Name**: Kaiwhakawānabe – The Couch Judge

## Core Features:

- User Onboarding & Profile Management: Enables user sign-up, secure login, age verification (including parental approval for minors), and personalization by selecting favorite teams, regions and kapa haka items to tailor their app experience. Also manages user data like critic points and vote/comment counts.
- Performance Stream & Details: Displays upcoming, live, and past kapa haka performances in a mobile-first, card-based interface. Each card shows key details like team name, event, and either a live video stream or performance image.
- Live Scoring & Voting System: During live performances, users can score individual kapa haka items (e.g., Whakaeke, Haka) using intuitive sliders. Submitting a vote updates fan score averages and vote counts in real-time.
- Fan Engagement (Comments & Social): Allows users to write comments, add emojis, and like comments on performances. Includes basic reporting features with a moderation queue for community safety.
- Critic Leaderboards: Showcases leaderboards (National, Regional, Event) ranking 'couch judge' based on the accuracy of their scores compared to official judging results, fostering friendly competition.
- Kapa Haka Judging Academy (AI-powered): A generative AI tool that provides detailed explanations, 'what to look for' criteria, and expert judging tips for each kapa haka performance item. This feature acts as an interactive learning resource, helping users understand scoring nuances based on comprehensive guidelines.
- Admin and Data Management: A simplified interface for administrators to create and manage events, add performances, upload official scores, and moderate user-submitted comments, ensuring data integrity and community standards.

## Style Guidelines:

- The application adopts a dark color scheme to emphasize video content and provide a modern, sophisticated aesthetic befitting kapa haka's cultural depth.
- Primary color: Vibrant OrangeRed (#FF4500). This bold hue is strategically used for interactive elements, calls-to-action, and key highlights to convey energy and cultural significance.
- Background color: A subtly warm, desaturated reddish-brown-grey (#0F0B0A). This deep background provides a richer canvas than pure black, allowing other elements to pop while maintaining a focus on performance content.
- Accent color: Pure White (#FFFFFF). Employed for primary text, essential icons, and prominent UI elements, ensuring optimal readability and a crisp, modern contrast against the dark background.
- Secondary UI color: Pure Black (#000000). Used for elements requiring maximum visual separation, such as strong dividers, modal overlays, or where absolute depth is desired in specific UI components.
- Universal font: 'Inter' (sans-serif). Selected for its exceptional legibility and modern, clean lines, making it suitable for all textual content, from headlines and scores to comments, across various mobile screen sizes.
- Utilize clean, vector-based iconography. Culturally relevant Māori-inspired motifs can be subtly integrated into appropriate icons (e.g., category indicators, team logos) to enhance the app's unique identity without sacrificing clarity.
- A mobile-first, card-based scrolling interface is central to the design. Large, clear cards for performances and events ensure easy readability and intuitive interaction. A consistent bottom navigation bar with 'Home', 'Events', 'Leaderboard', and 'Profile' tabs provides simple app navigation.
- Subtle and fluid animations are incorporated for user interactions such as swiping through cards, submitting votes, and real-time score updates. These micro-interactions enhance responsiveness and user feedback without being distracting.
